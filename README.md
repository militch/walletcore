# WALLET_CORE

区块链钱包跨平台通用库

## 集成方法

### JAVA 

Maven:
```xml
<dependency>
  <groupId>com.github.militch.walletcore</groupId>
  <artifactId>walletcorej</artifactId>
  <version>1.0.0</version>
</dependency>
```

Gradle:
```gradle
dependencies {
  implementation 'com.github.militch.walletcore:walletcorej:1.0.0'
}
```
